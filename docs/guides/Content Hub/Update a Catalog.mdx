You can modify the connector information and the field mappings for your created catalogs.

### Edit Catalog

1. From the Explore screen, next to each catalog name Click the ‘edit’ icon\*\* under the ”Actions” column.
1. Clicking Edit will take you to the Catalog configuration screen with the details pre-filled.

### Edit connector configuration

1. The configurations and credentials for the upload methods and connectors can be edited and updated.
1. Any changes made to the connection details, “Test Connection” operation must be successful in order for the changes to reflect

**Note**: If the Upload method configurations are changed, the connection must be tested in order to ‘Submit’ the Catalog.

### Modify Catalog Schema

1. Click on the fields that you want to delete/ modify.
1. Click on save once you are done.

**Note:**

- During the sync process, if new fields are identified in the modified catalog that are not present in the Catalog Schema, these fields will not be automatically added.
- You have to manually map the fields to its corresponding attribute.
- This process of adding the fields to the catalog schema can be done any time.

### Update Sync settings

1. You can modify the catalog sync settings at any time.
1. Once the changes are done, click on Next Step to Proceed to the Map Fields Step
1. Click on Submit

**Note**: The changes will be reflected only after submit is clicked. The Sync status will be reflected on the catalog listing screen.
