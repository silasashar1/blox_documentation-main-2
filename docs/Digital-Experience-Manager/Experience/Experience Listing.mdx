Experience created can be visualized in the Listing Table in the Experience Screen. The listing enables you to manage, search, sort, filter and delete Experience.

## Getting to Experience Listing

![Alt text](https://d1r1e7xjkfj7nz.cloudfront.net/experiencelisting1b.png 'a title')

**Navigation:**

- From the Vue Home Page -> Click on Digital Experience Hub to visit the Digital Experience Home Screen.
- Click on Experience on the navigation bar to access the Experience listing table.
- Alternatively, Hover over the Digital Experience Hub on the navigation bar and click on Experience.

### Search Experience

1. You can search for the created Experience using the Experience name via search bar or use one of the suggested/recently searched keywords.
1. You can find the search results populated in the table.

### Sort Columns

1. Hover over the column header to find the sort icon next to each column
1. Click the Sort icon to sort the column alphabetically either in ascending or descending order.
1. If the sort is applied to multiple columns, the column for which the sort was applied last will take precedence.

### Filter Columns

1. Hover over the column header to find the filter icon next to each column
1. Click the filter icon to open up a dropdown from which you can multi-select the values to be filtered.
1. You can also search for a filter value within the dropdown.
1. The table will be populated with the filtered results.

### Delete Experience

1. From the Experience listing table, next to each Experience name, click on the Delete icon\*\* under actions.
1. Clicking Delete will prompt the user with an overlay modal that lists all the entities (Pages) the Experience is linked with.

   2\.1. Here, you can access any entity config screen, make necessary changes before deleting the Experience

1. Deleting the Experience will unlink it from all the linked features and permanently delete it & the content from our system

### Publish / UnPublish

1. From the Experience listing table, next to each Experience name, click on the Publish  under actions to publish and unpublish an Experience
