# Embed code

## Integration

Before you can begin building personalized experiences you need to integrate your online store with the Vue app. This section helps you integrate Vue easily in a few short steps.

## Integration Via Embed Code

Embed code is a piece of code that you’ll add to your website’s pages to publish the experiences created using Vue. There are two ways in which the embed code can be put onto the website:

```
<script>
  (function(d, r, l) {
    const b = d.body;
    const s = d.createElement('script');
    s.defer = true;
    s.src = 'https://staging-Vue-ai-js.getblox.ai/1.0/Vue-ai.staging.js';
    b.appendChild(s);
    s.onload = function() {
      if (typeof window.blox_ai === 'object' && typeof window.blox_ai.init === 'function') {
        window.blox_ai.init(r, l);
      }
    };
  })(window.document, 'us-east-1', 'bloxDataLayer');
</script>

```

1. Launch Digital Experience Hub.
2. Click on the <button> Interegate </button> button the navigation bar
