


## Create a Module

To create a Module, follow these steps:

1. Click on the 'Create Module' Button on the Module Listing Screen. You will be brought to the Module Configuration Screen where you can select the integration mode, template, and link recommendations.

### Module Details

2. Click the 'Create Module' CTA on the Module Listing Screen.

3. Give your Module a unique name.

   - Note: You can save your Module config anytime by clicking on 'Continue Later' CTA & clicking on 'Save & Exit'. The Module is displayed on the listing screen in a draft state.
#### Integration Modes

You can select how you would like to integrate this module using the following options:

- **Embed Code**: Enables you to select a template, strategy(s), and define its configurations to render directly on your platform via Embed Code.

- **API**: Enables you to select strategy(s) and define its configurations to return results via API.

- **Email**: Enables you to configure an Email module by selecting an Email template, strategy(s), and define its configurations to return results via an HTML snippet.

   - Note: Changing the integration mode will reset the module configuration. This field cannot be changed while the Module is Live.

#### Select a Template

When the integration mode selected is Embed Code or Email, you can select a Template here to display your content within.

- Click the 'Select' button on the Select Template Field to view the Listing of created Templates.

- You will be able to view and scroll through the list of Created Templates to select from.

- The Desktop, Mobile, and Email (if applicable) Preview Each Template will be displayed on the right panel.

- You can use the Search or Filter by Template Type options to help narrow down your selection.

- Once you have selected your Template, click the 'Confirm' CTA to return to the Module Config screen where the selected template name will be displayed and its preview will appear on the right panel.

- You can edit your selection by clicking the 'Change' CTA.

   - Note: Changing the Template will reset the rest of module configuration. This field cannot be changed while the Module is Live.

## Module Configurations

You can define the properties of the module in this section. Module Configuration Options will vary based on the Integration Mode Selected.

#### Embed Code Module Configurations

- Module Title: Provide a Module Title in this section. Any text you input can be previewed on the Right panel Preview Section in the styling format of your selected Template.

- Min Tiles: Provide the minimum number of recommendation results you want returned to render the module on your platform.

- Max Tiles: Provide the maximum number of recommended results you want returned and displayed within this module. This field determines the number of Tiles in the module.
![Alt text](https://d1r1e7xjkfj7nz.cloudfront.net/module4.png 'a title')
#### API Module Configurations

- Min Tiles: Provide the minimum number of recommendation results you want returned to render the module on your platform.

- Max Tiles: Provide the maximum number of recommended results you want returned and displayed within this module. This field determines the number of Tiles in the module.

#### Email Module Configurations

- Module Title: Provide a Module Title in this section. Any text you input can be previewed on the Right panel Preview Section in the styling format of your selected Email Template.

- Email Platform: Select the Email Platform you wish to integrate this module into. This will help unify the HTML snippet format with the provided platform. If you would like to 'Add a New Platform', click the CTA and enter a Platform Name, User ID Format, Product ID Format, and Timestamp Format.

- Card Height and Width: Provide the desired Height and Width (in px) of each content Card. By adjusting the Height, the Width will automatically adjust to the desired ratio.

- Number of Rows and Columns: Provide the number of rows and columns you want to display within each email. These fields determine the number of Tiles in the module.

### Link Recommendations

Within the 'Link Recommendations' accordion, you will be able to select strategy(s) and/or static content to add to your module. You will also be able to assign the selected recommendations to specific tiles to meet your business needs.
![Alt text](https://d1r1e7xjkfj7nz.cloudfront.net/module5b.png 'a title')
### Select Recommendations

- Click on the 'Add Recommendations' CTA to open the Select Recommendations Modal.

- Select Strategy(s) from the first Tab, 'Strategy'. This Tab lists all your created Strategies. Use the Search, Filter, or Sort options to further narrow your selection. You can multi-select up to 3 strategies to add to your module.

- Select Content from the second Tab, 'Content'. Here you will find a listing of all your Content. You can navigate to different Catalogs or Feeds using the provided dropdowns. You can also use the Search, Filter, or Zoom options to further narrow your selection. Multi-Select as many Content(s) you wish to add to your Module.

- Once you have selected the desired strategies/content, click the 'Confirm' CTA, and you will see the selected strategies and/or content display within the Link Recommendations Accordion.

- To view the info, replace, or delete any of the linked recommendations, use the respective icons within the 'Actions' column.

- To add or edit your selection, click on the 'Manage Recommendations' CTA within the Link Recommendations Accordion to return the Select Recommendations Modal.

### Assign Tiles

Once you have linked recommendations to your module, you can assign them to specific tiles.

- To assign a linked Content, click on the dropdown associated with it under the column 'Tiles Assigned' and select the Tile where you would like that content to be placed. You can move it to any other available tile as needed. Once you are happy with the selection, click the 'Apply' CTA to save and close the dropdown.

- To assign a linked Strategy, click on the dropdown associated with it under the column 'Tiles Assigned' and select a tile or a range of tiles. To select a range, double click on the tile you want to start from and click on the end of your range to select all available tiles in between. You can also click the 'Select Unassigned' CTA to select all available tiles. Once you are happy with the selection, click the 'Apply' CTA to save and close the dropdown.

- To clear any selected tiles, you can use the 'Clear All' CTA on the column header to clear all selected tiles or you can clear the selection on any linked strategy or content within its Tiles Assigned dropdown.

- To see the number of tiles assigned in relation to the total, view the 'Assigned Tiles' progress bar. You can also view the count of each linked strategy/content on its assigned tiles dropdown.

#### Linking Recommendations - 'Actions' Column:

- Assign Fallback Strategy: In case a strategy returns fewer results than configured items, the fallback strategy will backfill recommended items. You can select one fallback strategy.

- Strategy/Content Details: If strategy linked, this option shows the information of the strategy linked, modules, experiences associated with the strategy. If content linked, it shows the content details - Metadata and Tags associated with the content.

- Replace: Allows to replace the content/strategy that has been linked.

- Remove: Allows to remove the content/strategy that has been linked.
